Build a modern web app called “Interview Ready AI”.

The app helps students assess their interview readiness in less than 2 minutes.

Core functionality:
1. Landing page with a clear headline:
   “Check Your Interview Readiness in 2 Minutes”
2. A simple form where the user enters:
   - Full name
   - Target job role
   - Experience level: Fresher, Intern, Entry-level
   - Resume text input box
   - Optional portfolio/GitHub/LinkedIn links
3. A button: “Analyze My Readiness”

After clicking the button, the app should analyze the resume/profile and generate:
- Overall Interview Readiness Score out of 100
- Readiness level:
  0–40 = Not Ready
  41–70 = Almost Ready
  71–85 = Interview Ready
  86–100 = Strong Candidate
- Category-wise scores:
  Resume Quality: 25 points
  Technical Skills Match: 25 points
  Project/Portfolio Strength: 20 points
  Communication & Presentation Readiness: 15 points
  Job Role Fit: 15 points

Evaluation logic:
Resume Quality should check formatting, clarity, grammar, presence of summary, education, skills, projects, achievements, and measurable impact.
Technical Skills Match should compare listed skills with the selected target role.
Project/Portfolio Strength should check whether projects include problem statement, technologies used, personal contribution, outcomes, links, and real-world relevance.
Communication Readiness should check whether the resume clearly explains achievements, avoids vague phrases, and presents information confidently.
Job Role Fit should check whether the resume is tailored to the selected role.

Output page should include:
1. A large circular score indicator
2. Category-wise progress bars
3. Top 3 strengths
4. Top 3 weaknesses
5. Specific improvement suggestions
6. A personalized 7-day improvement plan
7. Suggested interview questions based on weak areas
8. A “Download Report” button
9. A “Analyze Again” button

Design requirements:
- Clean student-friendly UI
- Modern dashboard layout
- Use cards, progress bars, badges, and icons
- Mobile responsive
- Use a blue, white, and purple color theme
- Keep the user flow very simple
- The analysis should feel instant

Tech requirements:
- Use React for frontend
- Use Tailwind CSS for styling
- Use mock AI analysis logic first, based on keyword and rule-based scoring
- Structure the code so that real AI API integration can be added later
- No login required for MVP

Create sample scoring logic:
- If resume includes skills relevant to the selected role, increase Technical Skills score
- If resume includes project descriptions, increase Portfolio score
- If resume includes numbers, achievements, or measurable impact, increase Resume Quality score
- If GitHub/LinkedIn/portfolio links are present, increase Portfolio score
- If resume lacks role-specific keywords, decrease Job Role Fit score
- If resume uses vague words like “hardworking” without proof, reduce Communication score

Also include sample target roles:
- Software Developer
- Frontend Developer
- Backend Developer
- Data Analyst
- Machine Learning Engineer
- UI/UX Designer
- Digital Marketing Intern

Make the final app feel like a working hackathon MVP that can be demonstrated in a 5-minute pitch.
